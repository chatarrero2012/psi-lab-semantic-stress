**Ψ LAB** is a specialized diagnostic environment designed to evaluate the integrity and "stress" levels of semantic streams. By utilizing vector embeddings and PCA (Principal Component Analysis), it maps how text evolves, drifts, or collapses under different conditions.

## 🧪 Core Diagnostics

- **Stability Index:** Measures the consistency of the semantic flow.
- **Semantic Drift:** Visualizes how meaning shifts over time in a 3D latent space.
- **Injection Scoring:** Real-time detection of adversarial patterns and prompt injection attempts.
- **Entropy Analysis:** Evaluates text randomness via "Semantic Temperature" controls.
